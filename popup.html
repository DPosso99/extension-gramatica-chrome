¿Cómo instalar el servidor?

1. Instala Java 11 o superior

2. Descarga LanguageTool (versión standalone):

languagetool.org/download

3. Descomprime y ejecuta:

java -jar languagetool-server.jar
--port 8081 --allow-origin "*"

El servidor queda disponible en
http://localhost:8081

⚠ El servidor debe estar corriendo cada vez que uses el corrector. Puedes crear un script .bat para iniciarlo rápido.
